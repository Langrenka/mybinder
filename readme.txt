В биндере откражены самые необходимые функции
Не стоит скидывать биндер тем, кто не в RRS Ghost
1) Распаковать архив в одну папку.
2) Запустить PoliceBinder.exe или PoliceBinder.ahk (если установлен AHK ANCI 32 бит)
3) Ввести ник, Звание, Тэг без скобок
4) Нажать сохранить настройки
5) Назначить клавиши
6) Нажать сохранить!!!!
7) Нажать старт
PS Биндер работает только свернутым в трей
F12 - перезагрузить
F11 - Развернуть биндер (работает не в игре)
Не забудь про настройки
Для загрузки лекций нужно закинуть в папку files/lectures файл .txt с лекцией
